Local Lossless WebP Converter

Anas Compression Script

Enter a folder path like C:\Users\bhagy\images. The tool will recursively find direct image files and image paths referenced inside CSV files. Direct image scans write sibling _lossless.webp files next to each source image. CSV-referenced images are grouped into the CSV file's subfolder.

Supports images found directly in the folder tree and image paths discovered inside CSV files.
